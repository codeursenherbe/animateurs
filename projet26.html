Projet 26 – Initier les élèves à l’IA avec Scratch et Teachable Machine

Cet atelier propose de faire découvrir l’intelligence artificielle aux élèves à travers un projet Scratch connecté à Teachable Machine : les enfants entraînent un modèle simple (reconnaissance de gestes ou d’objets), puis créent un mini‑jeu contrôlé par ce que « voit » l’IA (par exemple un jeu de type « Feu rouge / Feu vert » ou un quiz caméra). [web:21][web:59]

Les élèves ont déjà réalisé des projets Scratch (labyrinthe, mémory) et maîtrisent les bases : déplacements, collisions, variables simples. L’objectif est ici de les amener à comprendre qu’un programme peut apprendre à partir d’exemples et de réinvestir cette idée dans un jeu complet. [web:21][web:24]

Projet Scratch :

Ouvrir le projet (modèle de jeu IA) ou créer un nouveau projet Scratch et suivre les fiches « séances » pour connecter le jeu au modèle Teachable Machine entraîné par les élèves. [web:22][web:59]

Fiches détaillées :

  • Fiche séance 1 : Découvrir l’IA (jeu débranché « Qui est‑ce ? »), prise en main de Teachable Machine, création des classes et collecte des images avec la webcam. [web:21][web:59]
  • Fiche séance 2 : Entraîner le modèle, tester les prédictions, analyser les erreurs et exporter l’URL du modèle pour un usage dans Scratch. [web:21][web:47]
  • Fiche séance 3 : Créer un mini‑jeu Scratch (Feu rouge / Feu vert, quiz caméra, etc.) dont le comportement dépend de la classe prédite par l’IA, puis organiser des tests croisés entre binômes. [web:22][web:59]

Pour les animateurs débutants, il est conseillé de lire d’abord la fiche « Configurer Chrome et Teachable Machine » ainsi que de visionner un court tutoriel vidéo en français sur l’utilisation conjointe de Scratch et Teachable Machine, avant de lancer l’atelier avec les élèves. [web:39][web:40]
